Audio Story

This is an Audio story that I created for my first semester studying Interaction and Experience design at UL. The task was to tell a story using only sounds but wthout using dialogue or narration. I used Audacity to create this story and soundsamples found on freesound.org. You can find references for the sound clips on the readme file for this site.


Throughout this project, I learned about audio mixing, how to work with Audacity and how to be creative with sounds available to me.
